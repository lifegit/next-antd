## create-next-antd

这是一个基于 [Next.js 12](https://nextjs.org/blog/next-12) 、[react 18](https://beta.reactjs.org/) 、[antd](https://ant.design/index-cn) 开箱即用的ssr脚手架。

### 运行

```bash
npm run dev
# or
yarn dev
```

打开 [http://localhost:3000](http://localhost:3000) 查看页面.
